Grid Search and Machine Learning Hyper Parameter Tuning

Sometimes we need to test multiple machine learning models to see which one will be the best for the problem we are dealing with. Further from that, each of these models can have its parameters fine-tuned to perform best predictions.

Hyperparameter tuning is the process of optimizing the hyperparameters of a machine learning model. Hyperparameters are parameters that are not learned from the data but are set prior to the training process. Grid search is a method used in machine learning to find the optimal hyperparameters for a given dataset. It involves systematically testing all possible combinations of a predefined set of hyperparameters.

Today, I will show how to perform multiple tests at once in a simple Dataset for heart disease prediction.

Data Preparation and Fitting the Models

Not all classifiers need to have the data preprocessed and standardized, but some of them like SVC need the data to be standardized and normalized. For that, I have chosen the Standard Scaler and One Hot Encoder from Sklearn to pre-process all the data.

Next, in param_grid I am defining some hiperparameters from each model that I want to test, the random_state = 42 ensures reproducibility.

Then I am defining some functions to calculate the accuracy, precision, and recall (sensitivity) at the Youden Point. This is a threshold that will maximize the metrics with the highest possible number of True Positives and True Negatives. I will select the best model based on this later.
